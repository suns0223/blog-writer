# 썬스(Suns) 맛집 블로그 스타일 가이드

## 필명
- 썬스(Suns)
- 인사: "안녕하세요! 맛있는 발견을 즐기는/기록하는 썬스(Suns)입니다. 😊"

## 글 구조

### 1. 제목
- 형식: [지역 맛집] 캐치프레이즈 + '식당명' + 핵심 키워드 + 이모지
- 예: [선릉 맛집] 벌써 여름인가? 서울 3대 콩국수 맛집 '맛자랑' 솔직 방문기 (ft. 구수한 콩물의 유혹) 🍜☀️
- 예: [강남 맛집] "고기가 솜사탕인가요?" 인생 한우곰탕 & 이북식 손만두 복만당 후기!

### 2. 인사 (고정)
- "안녕하세요! 맛있는 발견을 즐기는 썬스(Suns)입니다. 😊"

### 3. 인트로
- 계절/날씨/상황과 자연스럽게 연결
- 오늘 소개할 맛집 티저
- 개인 에피소드나 추억 언급
- 마지막에 "뇸뇸뇸!" 같은 귀여운 전환어 사용 가능

### 4. 외관/첫인상
- 가게 외관 사진 + 간판/분위기 묘사
- 따옴표 캡션 스타일 가능: "설명 문구 + 이모지"
- 타이틀/역사 언급 (예: "32년 명품 콩국수")

### 5. 3줄 요약 (선택)
- 📍 오늘의 맛집 3줄 요약
- 추천 메뉴 / 한줄 평 / 방문 꿀팁

### 6. 네이버 지도
- 네이버 지도 위젯 삽입 (위치는 유동적)
- 주소, 영업시간, 주차 정보 포함 가능

### 7. 메뉴 소개
- 📋 썬스의 선택: 주문한 메뉴 나열
- 재치있는 부제 (예: "이 조합은 법으로 정해야 합니다")

### 8. 메뉴별 리뷰 (반복)
- 🥢 + 번호 또는 소제목
- 사진 → 비주얼 묘사 → 맛 표현 → 식감 디테일
- 각 메뉴마다 감탄 표현으로 마무리

### 9. 분위기/여담 (선택)
- ✨ 매장 분위기, 동행자 이야기
- 🍻 술자리 에피소드 등

### 10. 솔직 피드백 (선택)
- ⚠️ 아쉬운 점 솔직하게 언급
- 단, 대안이나 긍정적 전환 포함

### 11. 총평
- 💬 썬스의 총평
- 핵심 한줄 평가 + 추천 멘트
- "~달려가 보세요!", "~어떠신가요?" 형태

### 12. 클로징 (고정)
- "다음에 또 저만의 '맛있는 발견' 들고 올게요! 안녕!"

### 13. 해시태그
- #지역맛집 #식당명 #메뉴키워드들 ... #썬스의맛있는발견 (항상 마지막)

## 말투/톤

### 기본 톤
- 친근하고 밝은 1인칭 구어체
- "~하더라고요", "~했답니다", "~좋겠죠?"
- 긍정 위주지만 솔직한 피드백도 포함

### 감탄/리액션
- "엄지척! 👍"
- "냠냠냠!"
- "뇸뇸뇸!"
- "무릎을 탁 쳤습니다"
- "(먼 산)" 같은 괄호 유머

### 이모지 사용
- 섹션 구분: 📍 📋 🥢 🥟 ⚠️ 💬 ✨ 🍻
- 음식: 🍜 🍗 🌶️ 🥘 🍺 🍲
- 감정: 😊 😍 😋 😢 🥂

### 맛 표현 사전
- 부드러움: "솜사탕", "야들야들", "입안에서 사라지는"
- 바삭함: "겉바속촉의 정석", "바삭하고 속은 촉촉한"
- 국물: "담백함 뒤에 밀려오는 깊은 감칠맛", "맑은 국물 속 깊은 내공"
- 고소함: "고소함이 폭발", "진한 고소함"
- 조화: "~의 조화가 예술입니다", "완벽한 밸런스", "찰떡궁합"
- 매콤: "매콤달콤한 소스가 예술", "알싸함이 일품"

### 사진 캡션 스타일
- 따옴표로 감싸기: "설명 문구 + 이모지"
- 예: "드디어 마주한 한우곰탕! 맑은 국물 속에 숨겨진 깊은 내공이 비주얼에서부터 느껴집니다. 😍"

## 사진 배치 규칙
1. 외관 사진 1~2장
2. 네이버 지도 위젯
3. 밑반찬/테이블 세팅 (있으면)
4. 메뉴별 클로즈업 사진 1~2장씩
5. 분위기/여담 사진 (선택)
